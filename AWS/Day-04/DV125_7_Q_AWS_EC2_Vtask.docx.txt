2 How to terminate instances?
Ans. Select instances then go to action and click on terminate instances 
	
3 How to create launch configuration?
Ans. go to left panel and then click on launch configuration then fill details such as Name, AMI, instance type, select IAM instance profile and so on.
	
4 How to create auto scaling groups?
Ans.  Click to auto scaling group then fill the details such as auto scaling group name, select launch configuration, select a VPC, choose subnets, select load balancer, click on health check if you want, choose desired capacity (which is responsible for maintaining number of instances, if any terminated it will automatically launch another one with same script), add notification and add name tag.
