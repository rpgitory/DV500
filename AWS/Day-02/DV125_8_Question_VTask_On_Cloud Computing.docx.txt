Q4 What is the major service in AWS?
Ans AWS include Amazon EC2 (Elastic Compute Cloud) for virtual servers, Amazon S3 (Simple Storage Service) for object storage, Amazon RDS (Relational Database Service) for managed databases, AWS Lambda for serverless computing, and Amazon VPC (Virtual Private Cloud) for networking. These services form the foundation of many applications and solutions built on AWS.
	
Q5 EBS stands for?
Ans EBS stands for Elastic Block Store. It is a scalable block storage service provided by Amazon Web Services (AWS) that allows you to create and attach persistent block storage volumes to Amazon EC2 instances. EBS volumes provide durable and highly available storage for your EC2 instances, allowing you to store data persistently even after the instance is terminated.

Q8 What are the Google Cloud Platform?
Ans Google Cloud Platform (GCP) is a suite of cloud computing services offered by Google. It provides a wide range of infrastructure and platform services to help organizations build, deploy, and scale their applications and services in the cloud.
	
Q9 Full form of Azure VMs.
Ans Azure Virtual Machines is a service provided by Microsoft Azure that allows users to deploy and manage virtual machines in the cloud. It enables users to create and run virtual machines with different operating systems and software configurations, providing flexibility and scalability for various computing workloads.
	
Q10 Briefly explain the cloud computing v/s traditional design.
Ans traditional design need up-front cost, IT staff to manage servers, own security infrastructure, cooling system, backup and disaster recovery plans, max server capability, and it is hard to scalable horizontally where cloud computing do need up-front cost, no need IT staff to manage servers and all, no security problems, no worry about availability and fault tolerance, anytime scalable as you need, pay as you use.
