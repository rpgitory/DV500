Q1 WHAT IS REPLICATING OBJECTS IN AWS HOW ITS USEFUL?
ANS Replicating objects in AWS involves copying data from one AWS region or bucket to another. It is useful for data redundancy, disaster recovery, low latency access, compliance, load balancing, content distribution, data migration, cross-region collaboration, and multi-region disaster recovery. It ensures high availability, improves data access, and provides data resilience across different geographical locations.
