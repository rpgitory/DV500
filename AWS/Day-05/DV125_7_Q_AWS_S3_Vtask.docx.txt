Q1 WHAT IS AWS CLOUD FRONT?
ANS It is a globally-distributed network offered by AWS which is securely deliver content to the end user with high speed transfer rate and low latency.
	
Q2 Why AWS cloud front is useful?
ANS To provide new content with globally access and latency issues solved 
	
Q3 What are the benefits of cloud front?
ANS cost-effective, time saving, content privacy, highly programmable, geo-targeting, and quick content delivery.
	
Q4 How AWS could front delivers the content?
ANS client access website by typing URL, then client might request to access object to download then AWS cloud front will connect nearest aws server where file is stored.
